To run my project:

HTML:

All the HTML pages are in the src/main/resources/templates folder.

CSS:

There is only one CSS file within src/main/resources/static.css folder containing the one css file.

Client side js:

This is contained within the src/main/resources/static.js folder containing one js file that determines form validation and Review displaying.

Database write: the database writes the name, email, phone number and feedback when the user uses the review form or the contact us

Database read: the database reads the name, email and feedback when the query is run on the database and outputs it.


Security: there is a tls certificate and login required for accessing reviews.
